# Dark Patterns

## Overview
A set of dummy websites for a research about the influence of **dark patterns** on the User Experience (UX). 


## Installation
1. Clone this repo.
1. Run ```npm install```.
1. Run ```npm run dev```.
1. Point your browser to ```http://localhost:5173```.
